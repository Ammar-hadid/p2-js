Wat is het verschil met de script tag boven in te laden in de <head> tag t.o.v onder inladen aan het einde van je HTML tag

Door de script in de <head> te plaatsen stopt de browser met het laden van de html totdat de script is gedownload en uitgevoerd is.

Door het onderaan de HTML te plaatsen word het pas uitegvoerd nadat de HTML is geladen

Wat doet de attribuut “defer” in de <script> tag

'defer' zorgt ervoor dat de script in het achtegrond laadt terwijl de html nog aan het laden is en voert het pas uit als de HTML volledig geladen is.

Zonder 'defer' stopt the browser met het laden van HTML en laadt het het script en voert het gelijk uit

Wat is het verschil tussen HTML en JavaScript?

HTML is wat op de pagina staat, de structuur

JavaScript bepaalt wat de pagina doet, functionaliteit
